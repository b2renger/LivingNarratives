# LivingNarratives

Template folder for living narratives : here lives all the projects files and assets for production.

## Test the example project

Look at the Story_example.md

-> run the scripts


## Build your own

- Copy the Story_template.md, rename it with Story_your_project_name.md.
- Populate with your own story and assets, placing them in the assets folder.
- Fill in the descriptions and populate you story with your medias (descriptions, images, videos, audios).


-> run the scripts to test

-> run the scripts to export to the desired format
